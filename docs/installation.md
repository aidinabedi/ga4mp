# Loading the library

#### **ES6 Imports**
```javascript
import ga4mp from '@aidinabedi/ga4mp'
const ga4track = ga4mp(["G-THYNGSTER"], {
  user_id: undefined,
  non_personalized_ads: true,
  debug: true
});
```


```javascript
const ga4mp = require('./dist/ga4mp').default
```
